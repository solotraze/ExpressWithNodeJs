A simple Node.js web app using the Express framework (http://expressjs.com/).

Instructions:
1. After downloading the source, using shell type following commands:
    --------------------
    $cd <source_root>
    $npm install
    --------------------
  This will install dependency packages mentioned in the 'package.json' file.
2. Run following command:
    --------------------
    $node server.json
    --------------------
  This will start the HTTP server at the host and the port mentioned in the 'server.js' file. The shell will log the url of the started server. Note it.
3. Open browser with the link noted in the step 2.
